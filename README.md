Personal Expense Tracker

This project is a simple **Expense Tracking Application** built using Python.  
It allows users to add, view, and analyze daily expenses in an organized way.

 Features
- Add daily expenses with category and amount
- View list of all expenses
- Calculate total spending
- Stores data for future reference

 Technologies Used
- Python
- Jupyter Notebook

 How to Run
1. Download the project files
2. Open the notebook in **Jupyter Notebook**
3. Run all cells to use the tracker

 Future Enhancements
